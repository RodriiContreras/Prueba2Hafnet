Preguntas frecuentes

Finalicé mi pedido y olvidé ingresar el código de descuento.

Finalicé mi pedido y olvidé ingresar el código de descuento. Lamentablemente, una vez finalizado el pedido no es posible realizar ningún tipo de modificación. En este caso te recomendamos que solicites la cancelación de tu compra, y una vez confirmada la cancelación podrás realizar nuevamente el pedido aplicando el código correspondiente. Verifica que no hayas adquirido productos de oferta o promoción, ya que el descuento no aplica.

¿Cuánto tiempo tengo para realizar el pago si elegí hacerlo en efectivo?

Si al momento de realizar la compra elegiste abonar mediante PAGO FÁCIL o RAPI PAGO, una vez finalizado el pedido tenés 5 hasta días para concretar el pago, caso contrario se cancelará de manera automática cumplido dicho plazo.
